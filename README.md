# Uber-Clone
Uber clone is an on-demand taxi booking software that helps expand business reach by providing ride-hailing services to customers .

## Getting Started

- STEP: 1 Go to GCP console. 
- STEP: 2 Go to API & Keys.
- STEP: 3 Enable Google Map Key for Android.
- STEP: 4 Enable Google Map Key for iOS.
- STEP: 5 Create a Google Cloud Platform account and enable Billing through a mastercard or visa card for the project.
- STEP: 6 Create a Google Map Platform account and enable Billing through a mastercard or visa card for the same project.
- STEP: 7 Enable Places API.
- STEP: 8 Enable Direction API.
- STEP: 9 Copy the API key from Credtional section and paste it in Services -> map_config.dart file.
- STEP: 10 Paste the same Key in Android->app->src->main->AndroidManifest.xml file inside meta-data tag.
